# Test Scoreboard #
Provides a usage example of the library project [game-score-lib](https://github.com/naeemark/game-score-lib) 

---

### Input
```agsl
Argentina 0 - Australia 0
Uruguay 0 - Italy 0
Germany 0 - France 0
Spain 0 - Brazil 0
Mexico 0 - Canada 0
```

### Output
```agsl
Uruguay 6 - Italy 6
Spain 10 - Brazil 2
Mexico 0 - Canada 5
Argentina 3 - Australia 1
Germany 2 - France 2
```
